üöÄ Quick Start in 60 Seconds

Get your API key and make your first request:

bash 
curl https://api.blackroad.io/v1/status \
  -H "Authorization: Bearer YOUR_API_KEY"

Authentication

All API requests require authentication using API keys. Include your API key in the Authorization header:

http 
Authorization: Bearer YOUR_API_KEY

Getting Your API Key

1. Sign up at blackroad.io

2. Navigate to Settings ‚Üí API Keys

3. Generate a new API key

4. Copy and store it securely

Agents API Live

Manage and interact with 30,000 AI agents across your infrastructure.

GET /v1/agents

List all agents in your organization.

Query Parameters

Parameter Type Required Description
status string Optional Filter by status: active, idle, error
core string Optional Filter by Pi core: alice, aria, octavia, lucidia
limit integer Optional Number of results (default: 100, max: 1000)
javascript 
const response = await fetch('https://api.blackroad.io/v1/agents?status=active', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(`Active agents: ${data.agents.length}`);

Monitoring API Live

Real-time monitoring and metrics for your entire infrastructure.

GET /v1/metrics

Get real-time metrics for agents, services, and infrastructure.

python 
import requests

response = requests.get(
    'https://api.blackroad.io/v1/metrics',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

metrics = response.json()
print(f"CPU Usage: {metrics['cpu_usage']}%")
print(f"Active Agents: {metrics['agent_count']}")

WebSocket Streaming Live

Real-time bidirectional communication with sub-50ms latency.

javascript 
const ws = new WebSocket('wss://api.blackroad.io/v1/stream');

ws.onopen = () => {
  ws.send(JSON.stringify({
    type: 'subscribe',
    channels: ['agents', 'metrics']
  }));
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Real-time update:', data);
};
